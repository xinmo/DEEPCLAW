# 《产业研究室》小白操作手册

本文面向第一次使用《AI研究院 > 产业研究室》的用户编写。你不需要懂 AI、图谱、Agent 或监控规则，只要按本文一步一步操作，就可以完成一次基础的行业或个股研究。

## 1. 这个产品是干什么的

《产业研究室》是一个做产业链研究和个股上下游研究的工作台。你可以把它理解成一个“会自己查资料、整理证据、生成图谱、持续跟踪变化”的研究助手。

你可以用它做 4 件核心事情：

1. 研究一个行业，比如“AI 算力产业链”“机器人产业链”
2. 研究一只股票，比如“寒武纪”“中际旭创”
3. 自动生成产业链图谱，并查看节点、关系、证据
4. 把研究对象保存下来，后续持续跟踪变化、规则提醒和历史版本

## 2. 进入页面后你会看到什么

进入 `AI研究院 > 产业研究室` 后，页面大致分成 4 个区域：

1. 顶部
   这里有 `命令面板`、`刷新总览`、`新建研究对象`、`实时刷新` 开关

2. 左侧覆盖池
   这里会显示你已经创建过的研究对象，比如行业、个股

3. 中间主工作区
   这里有 3 个页签：
   - `深度研究`
   - `持续覆盖`
   - `图谱库`

4. 图谱区域
   图谱有 3 种查看模式：
   - `概览`
   - `探索`
   - `交互`

如果你是第一次使用，先不要管所有高级功能，按下面的“第一次完整使用流程”走就行。

## 3. 第一次使用的完整流程

第一次最推荐你做的是：

1. 新建一个研究对象
2. 运行一次深度研究
3. 看图谱
4. 点开一个节点看证据
5. 保存为后续持续覆盖对象

下面是详细步骤。

## 4. 第一步：新建研究对象

### 4.1 点击哪里

点击页面右上角的 `新建研究对象` 按钮。

### 4.2 会弹出什么

会弹出一个表单窗口，主要字段如下：

1. `对象名称`
   这是你这次研究的名字。
   例子：
   - `AI 算力产业链`
   - `寒武纪`

2. `对象类型`
   这里二选一：
   - `行业`
   - `个股`

3. `市场`
   默认填 `CN`，一般不用改。

4. `股票代码`
   只有做个股研究时建议填写。
   如果你研究行业，可以留空。

5. `核心命题`
   写一句你想研究什么。
   例子：
   - `我想看清 AI 算力产业链上下游结构`
   - `我想确认寒武纪的核心供应链和客户结构`

6. `关注点`
   写几个重点方向，用逗号隔开。
   例子：
   - `GPU, 光模块, 先进封装, HBM`
   - `供应商, 客户, 替代关系`

7. `标签`
   选填，方便后续搜索。
   例子：
   - `AI, 算力, 半导体`

8. `工作目录`
   新手可以直接留空。

9. `描述`
   选填。用于补充范围说明，比如“只看中国市场”。

### 4.3 你应该怎么填

如果你是第一次用，推荐这样填：

1. 对象名称：`AI 算力产业链`
2. 对象类型：`行业`
3. 市场：`CN`
4. 核心命题：`梳理 AI 算力产业链的核心环节和上下游关系`
5. 关注点：`GPU, 光模块, 先进封装, 服务器, IDC`

填好后，点击弹窗右下角的 `创建`。

## 5. 第二步：运行一次深度研究

创建成功后，页面会自动进入这个研究对象。

此时切到中间主区域的 `深度研究` 页签。

### 5.1 你会看到哪些控件

左侧 `研究控制台` 中通常会看到：

1. `模型`
   一般保持默认即可

2. `工作目录`
   新手通常不用改

3. `研究简报`
   这里写本次想让系统重点研究的内容

4. `附加关注点`
   这里写补充关键词，用逗号隔开

5. `启用 deepagents 实时运行时`
   这是一个开关

6. `执行深度研究`
   真正开始研究的按钮

### 5.2 新手建议怎么操作

推荐你这样用：

1. 在 `研究简报` 中输入：
   `请梳理 AI 算力产业链的核心环节、上游供应商、下游客户及关键关系`

2. 在 `附加关注点` 中输入：
   `GPU, HBM, 光模块, IDC`

3. `启用 deepagents 实时运行时`
   建议先打开
   如果你的环境暂时不稳定，也可以先关闭

4. 点击 `执行深度研究`

### 5.3 点击后会发生什么

系统会开始：

1. 采集资料
2. 抽取实体和关系
3. 生成图谱
4. 汇总证据
5. 保存一个版本

如果研究成功，你会看到：

1. 中间出现图谱
2. 图谱下方出现摘要和高亮标签
3. 右侧出现证据、执行轨迹、Agent 计划等内容

## 6. 第三步：怎么看图谱

图谱区域有 3 个按钮：

1. `概览`
2. `探索`
3. `交互`

### 6.1 概览模式

适合刚开始快速看全局。

你会看到：

1. `节点`
   表示图里有多少公司/环节/实体

2. `边`
   表示它们之间有多少关系

3. `真相状态`
   表示当前图谱里有没有待审核冲突

4. `核心实体`
   最重要的节点卡片

5. `环节分布`
   看产业链结构大概分成哪些部分

6. `高置信关系`
   可信度较高的重要关系

如果你什么都不懂，先只做一件事：
点击 `核心实体` 里的任意一个卡片。

### 6.2 探索模式

这是最适合小白上手的模式。

你会看到这些按钮或输入框：

1. `搜索框`
   用来搜节点名称、环节、股票代码

2. `全部节点`
   显示所有节点

3. 节点类型按钮
   比如行业、供应商、客户等

4. 方向筛选
   - `全部方向`
   - `向外`
   - `向内`

5. 跳数按钮
   - `1-hop`
   - `2-hop`

6. `固定当前节点`
   把当前节点固定下来，方便对比

7. `清空轨迹`
   清掉你点击过的节点导航轨迹

8. `清空固定`
   清掉已固定的节点

### 6.3 小白推荐操作法

如果你第一次看图谱，建议按这个顺序：

1. 点击 `探索`
2. 在左侧节点列表点一个你熟悉的公司
3. 看右边详情卡片
4. 点 `固定当前节点`
5. 再点另一个公司
6. 看右边的对比卡片

这样你就能快速比较两个节点周围的产业链关系差异。

### 6.4 右侧详情卡片怎么看

右侧一般有这些内容：

1. 节点名称
2. 节点类型
3. 节点摘要
4. 置信度
5. `可达节点`
6. `遍历关系`
7. `导出快照`

#### `导出快照` 按钮怎么用

如果你觉得当前这个局部图谱值得保存：

1. 先选中一个节点
2. 点击右侧卡片里的 `导出快照`
3. 系统会下载一个 JSON 文件

这个文件适合给技术人员或后续做进一步整理使用。

### 6.5 交互模式

如果你想拖动节点、缩放、自由看结构，就点击 `交互`。

适合：

1. 图谱已经生成好了
2. 你想看整体结构
3. 你想拖动节点位置

如果你的电脑一般，建议先用 `概览` 和 `探索`，需要时再用 `交互`。

## 7. 第四步：怎么看证据

在 `深度研究` 页签右侧有 `证据与执行` 卡片。

### 7.1 最简单的看法

1. 在图谱里点一个节点
2. 右侧会切换到这个节点的证据

你会看到：

1. 节点名称
2. 节点类型
3. 置信度
4. `证据`
5. `真相校验`
6. `Agent 计划`
7. `执行轨迹`

### 7.2 `证据` 区域怎么看

每条证据通常有：

1. 标题
2. 来源
3. 置信度
4. 抓取时间
5. 摘录内容

你不用一次看很多。
先看 2 到 3 条最相关的证据就够了。

### 7.3 `真相校验` 区域怎么看

这里主要是告诉你图谱有没有冲突。

常见信息：

1. `真相状态`
2. `规范实体数量`
3. `别名组数量`
4. `待审核数量`

如果下面出现冲突卡片，并且看到按钮 `处理`，说明系统发现了需要人工判断的地方。

#### `处理` 按钮怎么用

1. 点击冲突右侧的 `处理`
2. 弹窗会出现
3. 选择一个处理方式
4. 如果需要，选择 `规范节点`
5. 在 `备注` 里写你的判断理由
6. 点击 `提交处理`

如果你是小白，不确定时建议先：

1. 不处理
2. 或者写上备注
3. 让更懂业务的人复核

## 8. 第五步：怎么做持续覆盖

切到中间主区域的 `持续覆盖` 页签。

这里是用来长期跟踪一个行业或个股变化的。

### 8.1 覆盖刷新怎么用

在左侧 `覆盖刷新` 卡片中：

1. 先选 `影响等级`
2. 在大文本框里描述最近发生的变化
3. 在 `附加跟踪点` 里写补充关注项
4. 点击 `刷新持续覆盖`

### 8.2 新手建议怎么写

你可以这样写：

1. 影响等级：`中影响`
2. 描述：
   `最近国内算力需求上升，服务器和光模块厂商订单可能变化，请更新产业链关系和重点公司影响`
3. 附加跟踪点：
   `订单, 客户, 价格, 产能`

### 8.3 覆盖刷新后看哪里

刷新完成后，重点看 3 个地方：

1. `变化流`
   看有哪些新事件、新变化

2. `提醒、规则与计划`
   看有没有触发提醒

3. `图谱库`
   看是否形成了新版本

## 9. 第六步：怎么设置提醒规则

如果你希望系统长期帮你盯某些变化，就在 `持续覆盖` 页签里点击：

1. `新建规则`

### 9.1 小白只需要填哪些

如果你不熟悉规则系统，先只填这 5 项：

1. `规则名称`
2. `规则类型`
3. `作用范围`
4. `关键词`
5. `严重级别`

### 9.2 最简单的例子

比如你想盯“价格变化”：

1. 规则名称：`HBM 价格波动`
2. 规则类型：`价格监控`
3. 作用范围：`产业链`
4. 关键词：`涨价, 降价, 价格, 供需`
5. 严重级别：`警告`

然后点击 `创建`。

### 9.3 哪些字段可以先不填

以下字段偏高级，你完全可以先不动：

1. `搜索范围`
2. `证据来源类型`
3. `节点类型`
4. `关系类型`
5. `冲突类型`
6. `冲突严重级别`
7. `最低置信度`

如果你是小白，这些先留空没有问题。

## 10. 第七步：怎么设置自动执行计划

如果你不想每次手动刷新覆盖，可以创建 `覆盖计划`。

在 `持续覆盖` 页签右侧点击：

1. `新建计划`

### 10.1 最简单的填法

1. 计划名称：`每日盘前覆盖`
2. 执行频率（分钟）：`1440`
3. 影响等级：`中影响`
4. 关注点：`订单, 价格, 客户结构`
5. 保持 `启用`

点击 `创建` 后，系统会把这个研究对象加入定时覆盖。

### 10.2 页面上怎么看是否生效

看 `覆盖计划` 列表里的这些信息：

1. `计划名称`
2. `执行频率`
3. `下次`
4. `最近任务`

如果你想立刻执行，不等下次时间：

1. 回到左侧 `覆盖刷新` 卡片
2. 点击 `立即执行到期计划`

## 11. 第八步：怎么查看历史版本

切到 `图谱库` 页签。

这里是你所有历史研究结果的版本中心。

### 11.1 左侧能看到什么

1. `版本历史`
2. `时间线`

### 11.2 你怎么操作

1. 点击左侧任意一个版本
2. 中间会切换到该版本图谱
3. 右侧会出现该版本分析

### 11.3 右侧可以做什么

你可以：

1. 查看版本摘要
2. 查看版本节点数和边数变化
3. 看冲突情况
4. 导出节点 CSV
5. 导出边 CSV
6. 比较两个版本差异

## 12. 第九步：怎么比较两个版本差异

在 `图谱库 > 版本分析` 中：

1. 找到 `图谱差异`
2. 选择 `基准版本`
3. 选择 `目标版本`

系统会自动显示：

1. 新增节点
2. 移除节点
3. 节点变更
4. 边变更

如果你想下载：

1. 点击 `导出 Diff Markdown`

## 13. 常用按钮速查表

### 顶部按钮

1. `命令面板`
   打开快捷操作菜单

2. `刷新总览`
   重新加载页面数据

3. `新建研究对象`
   新建行业或个股研究项

4. `实时刷新`
   打开后，后台活动会自动刷新到页面

### 深度研究页签

1. `执行深度研究`
   开始一次完整研究

2. `导出快照`
   导出当前图谱局部视图

3. `处理`
   处理图谱冲突

### 持续覆盖页签

1. `刷新持续覆盖`
   手动更新当前研究对象

2. `评估监控规则`
   预览当前规则是否触发

3. `立即执行到期计划`
   立刻运行已到期的计划任务

4. `新建规则`
   创建提醒规则

5. `新建计划`
   创建定时覆盖计划

### 图谱库页签

1. `导出节点 CSV`
2. `导出边 CSV`
3. `导出 Diff Markdown`

## 14. 快捷键怎么用

页面支持这些快捷键：

1. `Ctrl/Cmd + K`
   打开命令面板

2. `Ctrl/Cmd + 1`
   切到 `深度研究`

3. `Ctrl/Cmd + 2`
   切到 `持续覆盖`

4. `Ctrl/Cmd + 3`
   切到 `图谱库`

如果你不习惯快捷键，完全可以只用鼠标点按钮。

## 15. 新手最推荐的使用顺序

如果你完全不熟，建议永远按下面顺序走：

1. 点击 `新建研究对象`
2. 填对象名称、类型、核心命题、关注点
3. 点击 `创建`
4. 切到 `深度研究`
5. 写 `研究简报`
6. 点击 `执行深度研究`
7. 在图谱里切到 `探索`
8. 点击一个节点看右侧证据
9. 切到 `持续覆盖`
10. 创建一个简单规则
11. 创建一个简单计划
12. 切到 `图谱库` 看历史版本

## 16. 小白常见问题

### 16.1 页面里选项很多，我会不会填错

不会。你只要先填最基础的几个字段：

1. 名称
2. 类型
3. 核心命题
4. 关注点

其他高级字段可以先不动。

### 16.2 为什么图谱里有些词还是偏技术

因为有一部分内容来自系统自动抽取的实体类型、关系类型或后端结构字段。你先用中文按钮和中文区域理解流程即可，不需要一次看懂所有专业词。

### 16.3 我看不懂图谱，怎么办

先不要用 `交互`。
先用 `概览`，再用 `探索`。

最简单方法：

1. 点一个你熟悉的公司
2. 看右侧摘要
3. 看 `可达节点`
4. 看 `遍历关系`

### 16.4 我担心误操作

最安全的操作是：

1. 只新建对象
2. 只运行深度研究
3. 只查看图谱
4. 不处理冲突
5. 不创建复杂规则

这样基本不会有风险。

## 17. 一句话总结

如果你只记住一句话，就记住这个：

先 `新建研究对象`，再点 `执行深度研究`，然后在图谱里切到 `探索` 去点节点看证据；等你熟了，再去用 `持续覆盖`、`规则`、`计划` 和 `图谱库`。
